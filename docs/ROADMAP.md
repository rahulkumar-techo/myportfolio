# Roadmap

These are planned or suggested enhancements for the project.

## Near-Term

- Unify authentication strategy (NextAuth vs custom JWT).
- Standardize API responses across all routes.
- Add schema validation everywhere using Zod.
- Expand automated tests (unit + API).
- Improve admin analytics dashboards.

## Notifications

- Add scheduling for notification broadcasts.
- Add segmentation by tags and categories.
- Add retry queue for failed email sends.
- Add visibility into push delivery success.

## Platform

- CI pipeline for lint/test/build.
- Dockerized dev and production images.
- Better content versioning for blog and projects.
- Image optimization and caching rules for external SVGs.
